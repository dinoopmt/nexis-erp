HOW IT WORKS:
═══════════════════════════════════════════════════════════════════════

Edit GRN-1 from 100 → 150 units:

    ┌─────────────────────────────────────────────────┐
    │ START TRANSACTION 🔄                            │
    ├─────────────────────────────────────────────────┤
    │ 1. Calculate delta: 150 - 100 = +50             │
    │ 2. Update CurrentStock: $inc: +50               │
    │    350 → 400 ✅                                 │
    │ 3. Recalc availableQty: 400 - 0 - 0 = 400       │
    │ 4. Update InventoryBatch: qty → 150             │
    │ 5. Log StockBefore: delta = +50                 │
    │ 6. Update VendorPayment (if PENDING)            │
    │ 7. Update GRN: qty → 150                        │
    ├─────────────────────────────────────────────────┤
    │ COMMIT TRANSACTION 🎯                          │
    └─────────────────────────────────────────────────┘

    If ANY error → AUTOMATIC ROLLBACK (all changes undo)
