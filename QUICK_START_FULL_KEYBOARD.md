## 30-Second Tutorial

### Step 1: Search & Add
```
Type "Product ABC"
Press Enter
→ Item added, Qty field focused (text selected)
```

### Step 2: Edit Qty
```
See selected text in Qty field?
Type "5" (replaces selected text)
Press Tab
→ Price field now focused (text selected)
```

### Step 3: Edit Price
```
Type "99.99"
Press Tab
→ Discount % now focused
```

### Step 4: Edit Discounts
```
Type "10" for discount %
Press Tab
→ Discount Amount focused
Leave as "0" or type amount
Press Tab
```

### Step 5: Next Item or Save
```
If more items:
  Tab takes you to Item 2's Qty field
  Repeat from Step 1

If done:
  Press Ctrl+S to save
  Or Ctrl+P to print
```

## Most Important Shortcuts

### The 5 Keys You NEED to Know:

| Key | What It Does | When to Use |
|-----|------|-----------|
| **Tab** | Move to next field | After typing in any field |
| **Enter** | Add item / Next field | 1) Search: add item, 2) Table: next field |
| **Delete** | Remove item | Want to delete current row |
| **Ctrl+S** | Save invoice | When done entering all items |
| **Escape** | Back to search | To reset or start new section |

That's it! Learn these 5, you're good to go.

## Common Mistakes & Fixes

| What Went Wrong | Solution |
|---|---|
| Tab doesn't work | Make sure you're in a number field (Qty, Price, etc) |
| Focus jumped | Press Escape to reset and go back to search |
| Deleted by accident | Use Ctrl+Z (undo) if available |
| Ctrl+S didn't save | Check if dialog appeared - click OK |
| Can't find Delete key | Look for Delete key on keyboard (top right section) |

## Still Confused?

**Watch this mental model:**

Imagine the invoice form is like a spreadsheet:

```
┌──────┬───────┬──────────┬────────┬───────────┐
│ Item │ Qty   │ Price    │ Disc%  │ DiscAmt   │
├──────┼───────┼──────────┼────────┼───────────┤
│ Row1 │[Qty]→ │ Price→   │ Disc%→ │ DiscAmt→  │
├──────┼───────┼──────────┼────────┼───────────┤
│ Row2 │ Qty   │ Price    │ Disc%  │ DiscAmt   │
├──────┼───────┼──────────┼────────┼───────────┤
│ Row3 │ Qty   │ Price    │ Disc%  │ DiscAmt   │
└──────┴───────┴──────────┴────────┴───────────┘

Tab goes →  (next column)
Enter goes →  (next column)
↓ goes ↓   (rows down, same column)
↑ goes ↑   (rows up, same column)
```

That's it! Standard spreadsheet navigation.

## When to Use Each Key

| Situation | Use This | Result |
|-----------|----------|--------|
| Finished editing Qty | Tab | Go to Price |
| Need to fix previous field | Shift+Tab | Go back |
| Want to edit item above | ↑ | Move up, same column |
| Want to edit item below | ↓ | Move down, same column |
| Want to delete this item | Delete | Item gone |
| Want to save work | Ctrl+S | Saved! ✓ |
| Want to print | Ctrl+P | Opens print |
| Got lost | Escape | Back to search |
